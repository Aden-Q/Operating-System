# kinfo

Function: get system information from `/proc` file system

Info type:

+ CPU type
+ Kernel version
+ System operating time
+ CPU stat
+ Memory information
+ System average load

Usage:

```c
$ gcc -w kinfo.c -o kinfo
$ ./kinfo
```
